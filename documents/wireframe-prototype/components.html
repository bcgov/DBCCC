Layers overlay

This allows for adjusting:

  • Base layers (here assumed to be the same as Google Maps - Map, Terrain, and Satellite
  • Map features (here, as indicated in user story, presense of a web cam and presence of a road event)

Base layers toggle

This allows mutually exclusive toggle between:

  • Map
  • Terrain
  • Satellite


Assumption here is that this is available through Google Maps, base layer to swap between don’t appear to be indicated in the requirements.

3

Map feature checkboxes

Each checkbox will toggle visibility of the corresponding feature sets, namely:

  • Presence of a webcam
  • Presence of a road event


NOTE: Webcam is an existing feature on DriveBC

NOTE: Road event is not an existing feature on DriveBC, but could be any of the following:

  • Road condition
  • Incident
  • Current planned event
  • Future planned event
